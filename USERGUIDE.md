# Yuukei をはじめる

Yuukei(ゆうけい)は、あなたのデバイスに「住人」が住みつくアプリです。住人はデスクトップで生活し、話しかけたり、触ったり、あなたの帰りを待ったりします。ウィンドウは住人にとっての地形で、フォルダは部屋です。

この文書は、Yuukeiを使う人のための案内です。自分の住人、台本、拡張機能を作りたい人は [ユーザー・作者ガイド](docs/user/README.md) から始めてください。

## 1. インストール

正式なインストーラは準備中です(β配布はGitHub Releasesで行う予定)。現在はソースからの起動になります。

必要なもの: [Node.js](https://nodejs.org/)(v20以降)、[pnpm](https://pnpm.io/)、[Rust](https://rustup.rs/)。

```sh
pnpm install
pnpm dev:tauri
```

しばらく待つと、デスクトップに住人が現れます。

## 2. 初回起動

初回はオンボーディング(はじめての案内)が開きます。3つのステップがあります。

1. **ようこそ** — 使うWorld Pack(住人と世界のセット)を確認します。そのまま進めば標準の住人(ゆうけいとパートナー)です。
2. **AI(ことば)の設定** — AIをつなぐと、住人が台本にない言葉にも答えるようになります。**AIがなくても住人は生活します**。「AIなしで始める」を選べば、この設定は飛ばせます(あとから変えられます)。
3. **観測とプライバシー** — 住人がデスクトップの出来事(ウィンドウ・フォルダ・ダウンロード)に気づくための観測です。**すべて初期状態ではOFF**です。何を記録し、何を記録しないか(ウィンドウのタイトルやファイルのフルパスは記録しません)が画面に書いてあります。

「あとで」を押すかウィンドウを閉じた場合も、次回起動時にオンボーディングが自動で開くことはありません。設定はいつでもトレイ(メニューバー)アイコンの「設定」から変えられます。

## 3. 住人とのかかわり方

- **話しかける** — 入力欄から話しかけると、住人が答えます(台本にある話題は台本で、それ以外はAI設定があればAIで)。
- **触る** — 住人をクリックしたり、なでたりすると反応します。頭・服・髪で反応が変わります。
- **選択肢** — 住人が問いかけてきたときは、バルーンのボタンで答えられます。放っておくと、住人は静かに引き下がります。
- **出かける・帰る** — しばらく操作しないと住人はお留守番になり、戻ってくると出迎えてくれます。
- **観測をONにすると** — 新しいウィンドウの縁に座ったり、Downloadsフォルダであなたが最近ダウンロードしたものに反応したりします。

## 4. AI(LLM)の設定

設定画面の拡張(yuukei-intelligence)から設定します。

- **LM Studio などのローカルLLM** — 「OpenAI互換」を選び、Base URL(例: `http://127.0.0.1:1234/v1`)とモデル名を入れます。
- **Gemini** — 「Gemini」を選び、APIキーを入れます。

AIの接続先や使用量は設定画面で確認できます。AIが応答しなくても、住人の生活は台本で続きます。

## 5. 声(VOICEVOX)の設定

[VOICEVOX](https://voicevox.hiroshiba.jp/) を起動しておくと、住人のセリフが声になります。

1. VOICEVOXをインストールして起動する(既定で `127.0.0.1:50021` で待ち受けます)。
2. Yuukeiの設定画面でyuukei-voicevox拡張を有効にする。
3. 話者や話速は拡張の設定から変えられます(住人ごとに別の声にできます)。

VOICEVOXが起動していないときは、声は出ずに文字だけになります(エラーにはなりません)。VOICEVOXで合成した音声の利用規約は、各キャラクターの規約に従ってください。

## 6. World Packの導入(住人の交換)

World Packは住人・見た目・台本のセットです。

- **zipから** — 設定画面のWorld Pack欄「zipから読み込む」でzipを選びます。配布条件(ライセンス)が表示されるので、確認してから取り込みます。
- **フォルダから** — 展開済みのWorld Packフォルダを直接選ぶこともできます。

取り込んだPackごとに住人の生活史(記録・記憶・変数)は別々に保存されます。

## 7. 記録とプライバシー

Yuukeiは住人との生活を「生活の記録(イベントログ)」として端末内に保存します。外部に送信されるのは、AI設定で自分でつないだ先(LM StudioやGemini)への会話文脈だけです。

- **観測のON/OFF** — 設定画面「観測とプライバシー」でいつでも変えられます。
- **記録の確認と削除** — 設定画面「生活の記録」で内容を確認し、期間指定・種類指定・全削除ができます。
- **住人の記憶** — 記録から作られた「記憶」(要約)は「記憶」タブで見る・直す・忘れさせることができます。

## 8. こまったとき

| 症状 | 確認すること |
| --- | --- |
| 住人が現れない | 設定画面のWorld Pack欄に読み込みエラーが出ていないか |
| 声が出ない | VOICEVOXが起動しているか。拡張が有効か |
| AIの返事がない | 接続先(LM Studio等)が起動しているか。APIキーが正しいか。AIなしでも生活は続きます |
| フォルダやウィンドウに反応しない | 「観測とプライバシー」がONか。macOSでは初回にアクセシビリティ/オートメーションの許可を求められることがあります |
| 動きがおかしい | いったん終了して起動し直す。直らなければ「生活の記録」から直近の記録を確認 |
